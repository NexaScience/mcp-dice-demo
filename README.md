# GENERATED_SERVER_NAME

A Streamable HTTP MCP server generated by [`NexaScience/mcp-generator`](https://github.com/NexaScience/mcp-generator).

## Run locally

```bash
npm install
npm run build
npm start   # listens on 0.0.0.0:$PORT (default 3000), POST /mcp
```

## Deploy to Alpic

One click: open
`https://app.alpic.ai/new/clone?repositoryUrl=<this repo url>`.
Alpic auto-detects the `build` (tsc) and `start` (node dist/index.js) scripts; no
`alpic.json` or Dockerfile is required.
